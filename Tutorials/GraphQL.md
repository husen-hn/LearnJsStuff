# آموزش اولیه GraphQL
در این آموزش که از یودمی تهیه شده است بسشتر صحبت روی بخش جاوااسکریپیتی GraphQL می باشد و یک سرور با node و اکپسرس برای دریافت و ارسال داده بالا می آورد و هدف نهایی نیز ReactJS می باشد. برای شروع اولیه این آموزش لطفا مطالب جاوااسکریپت و ری اکت را مطالعه کنید.
